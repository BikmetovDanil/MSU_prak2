Задание практикума модельный веб-сервер

Добро пожаловать на сайт!

Первый этап

Сервер принимает запрос и отдаёт файл

Также сервер передаёт некоторую информацию о файле (размер и т.д.)

При ошибках 404, 501 появляется сообщение для пользователя

Второй этап

Сервер работает на системном вызове select

Появилась возможность обрабатывать cgi-сценарии

Формируются переменные окружения, вывод перенаправляется в файл

При успешном завершении содержимое файла выводится на экран
